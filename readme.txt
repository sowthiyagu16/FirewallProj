# FirewallObserver Project Overview

## Setup

Unzip the **`FirewallObserver.zip`** file to a directory, for example:

    C:\FirewallObserver

The package contains three projects:
------------------------------------------------------------------------

## 1. FirewallObserver

**FirewallObserver** is a C++ application designed to monitor the
Windows Firewall status and verify the integrity of its log files using
**TPM-based signature verification**.

-   The `FirewallService` class provides APIs to start and stop the
    firewall status monitor for all three profiles: **Domain**,
    **Private**, and **Public**.

-   Log files are written to a static path:

        C:\Firewall_monitor_logs

-   Each remediation report is saved as a `.log` file, and its
    corresponding TPM signature is stored as a `.sig` file.

    **Note:** This application will be executed in elevated mode.

------------------------------------------------------------------------

## 2. FirewallMonitorServiceClient

**FirewallMonitorServiceClient** is a Rust application that monitors
Windows Firewall status using a C++ backend via **CXX FFI bindings**.\
It logs firewall events and signs them using the **Trusted Platform
Module (TPM)**.

### Build

``` bash
cargo build
cargo build --release
```

### Run

``` bash
cargo run
```

**Executables:**

    .\target\debug\FirewallMonitorService.exe
    .\target\release\FirewallMonitorService.exe

------------------------------------------------------------------------

## 3. TPMSigVerifier

**TPMSigVerifier** validates the TPM signatures generated by the
`FirewallMonitorServiceClient`, ensuring the authenticity and integrity
of the logged reports.

### Build

``` bash
cargo build
cargo build --release
```

### Run

``` bash
cargo run -- -l "C:\Firewall_monitor_logs\Report_2025-11-06_08-32-26.log" -s "C:\Firewall_monitor_logs\Report_2025-11-06_08-32-26.sig"
```

**Executables:**

    .\target\debug\TPMSigVerifier.exe
    .\target\release\TPMSigVerifier.exe
